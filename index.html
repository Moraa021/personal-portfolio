City Services App - Case Study

Project Overview

The City Services App was designed to provide residents with intuitive access to urban services through a centralized digital platform. This project involved creating comprehensive wireframes for all key user flows.

Challenge

City residents were frustrated with fragmented services across multiple platforms, leading to confusion and inefficiency in accessing basic urban services like waste collection, parking permits, and utility payments.

Solution

I designed a unified interface that:

  • Consolidated all city services into a single platform
  • Implemented intuitive navigation with clear categorization
  • Created responsive designs for both mobile and desktop users
  • Incorporated accessibility features for diverse users

Process

  1. Conducted user research through surveys and interviews
  2. Created user personas and journey maps
  3. Developed low-fidelity wireframes for core flows
  4. Iterated based on stakeholder feedback
  5. Delivered high-fidelity wireframes with detailed annotations

Results

The wireframes received distinction from faculty and were praised for their:

  • Clear information architecture
  • Intuitive navigation patterns
  • Comprehensive coverage of user needs

Restaurant Queue System - Case Study

Project Overview

The Restaurant Queue System was designed to streamline customer flow and reduce wait times through a digital queuing solution. This project focused on creating an efficient user journey from arrival to seating.

Challenge

Popular restaurants were losing customers due to long wait times and inefficient manual queue management. Customers had no visibility into their wait status and often left before being seated.

Solution

I designed a system that:

  • Allowed digital check-in via mobile or kiosk
  • Provided real-time queue position updates
  • Enabled SMS notifications when table is ready
  • Gave restaurants tools to manage seating efficiency

Process

  1. Observed queue management at 3 different restaurants
  2. Identified pain points through customer interviews
  3. Created flow diagrams for all user scenarios
  4. Developed interactive wireframes for key screens
  5. Conducted usability testing with prototypes

Results

The solution demonstrated potential to:

  • Reduce customer walkaways by 40%
  • Improve table turnover efficiency
  • Enhance customer satisfaction with transparency
